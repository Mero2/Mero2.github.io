老夫只用jQuery（一）

发表于 2018-08-19 | 分类于 学习笔记 | 评论数： | 阅读次数：

jQuery

（1）jQuery是一个非常优秀的js库，九千行代码，重在封装思想，使用方法会用就可以，可以现查现用，zepto（移动端）从jQuery演变而来。
（2）jQuery官网下载jQuery-3.2.1.js，通过<script src = "jquery-3.2.1.js"></script>
（3）jQuery选中返回的是jQuery对象，而原生返回的dom结构，两者不可以混用。对象可以通过点儿什么来调用方法

$(“li”)的简单内部原理

$(null)不会返回jQuery对象，但也不会报错

$('#demo').css({'background':'red','width':100,'height';100})写出一个宽高为100 红色方块儿。

jQuery选择元素

（1）$()；里面和css选择一样，注意对选择出的一组元素，一起处理，省略循环，这在js当中是不允许的。
（2）括号里面可以写css选择器，原生dom，jQuery对象，null，undefined，函数，selector、content等。
（3）jQuery特有的选择规则（选择出来的是jQuery对象）$(‘ul>li:fist/last’) ;$(‘ul>li:eq(2)’);$(‘ul>li;odd/even’)选出奇数和偶数 $(‘li’,’ul’) 选出ul下的li。

小练习

window.onload先执行完毕，$()才触发，即先打印9999

过滤所有函数返回true的index值，filter改为not，就是选出来的不进行改变

$()

（1）$('li').has('div') 有div的li。返回li，后面操作的为li
$('li').find('div') 返回div，操作为div
（2）is()判断前后选出的是否是一个dom，返回true或false
console.log($('li:eq(0)').is('.item'));
（3）$('li').html('<p>222</p>'); 选出li，添加<p>222</p>的dom结构
$('li').html(); 括号里没有值得时候，取出第一个dom结构
$('li').text(); 无值的时候，取的是li下的所有的文本
（4）点击事件

$(‘li’).addClass(function(index) {
if(index%3 ===0) {
return ‘li-3’;
}else{
return ‘no-li-3’;
}
});
$(‘li’).removeClass(function(index) {
return ‘no-li-3’;
})

$(‘.box1’).appendTo($(‘.content’));
$(‘.box2’).appendTo($(‘.content’));
$(‘.box1’).on(‘click’,function() {
a = $(this).remove();
})
$(‘.box2’).on(‘click’,function() {
b = $(this).detach();
})
$(‘.box’).on(‘click’,function() {
a.appendTo($(‘.content’));
b.appendTo($(‘.content’));
})
`
上述代码，点box1，box2，分别消失，点box，出现box1 ，box2，当再次点击remove删除的box1时，没有点击事件，也就没法再次删除；

（9）创建dom：$('<div></div>'),$('<div>'),$('div/') 都能创建dom
