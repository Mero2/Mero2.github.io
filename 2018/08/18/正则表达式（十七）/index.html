正则表达式（十七）

发表于 2018-08-18 | 分类于 学习笔记 | 评论数： | 阅读次数：

正则表达式的作用

匹配特殊字符或者有特殊搭配原则的字符的最佳选择，比如用户名或者密码的格式进行输入限制。

转义字符

（1）转义字符 “\” ：当字符串中存在不符合格式的文本，但又要进行显示时，用转义字符，比如当字符串中间有一个引号时，无法进行识别，要报错，此时在引号的前面加转义字符就能实现你的想法。\ + 特定字符代表不同的功能，\t 代表tab键，\r代表行结束符，\n代表换行。
（2）多行字符串：当写的字符串需要在引号内进行换行时，在每行的末尾加上转义字符。才符合格式规定。

创建正则表达式的方法

（1）对象直接量的方式进行创建：var reg1 = /abc/; 匹配字符串中是否含有abc的片段。
（2）正则表达式也是一个对象，var reg = new RegExp(“abc”);匹配字符串中是否含有abc的片段。
如果不用new运算符，而是将RegExp()作为函数调用，那么它的行为与用new运算符调用时一样。

但是当reg改变时，reg1也会改变，拿到的只是reg的引用。

正则的三个修饰符

（1）i：执行对大小写不敏感的匹配（ignore不区分大小写）
（2）g：执行全局匹配（global查找所有的匹配项而非找到第一个匹配项后停止）
（3）m：执行多行匹配（multiple当字符串中有\n换行时，匹配以什么开头和以什么结尾时，这时看成两行的字符串的形式）

常用的正则匹配

单独的括号没什么意义，中间的竖线是或者的意思，括号也是一种运算，var reg = /(abc|bcd)d/g：代表在全局匹配字符abcd或者bcdd。
var str = "bb";var reg = /(\w)\1;
此处的括号代表一个字表达式，后面的\1代表引用第一个子表达式匹配到的内容，此段代码最终匹配到的是两个相同字符相连。

正则表达式的方法

（1）查看匹配的字符：var str = “字符”；var reg = /正则表达式/；str.match(reg);查看匹配到的str里的字符。
（2）看是否匹配到字符：var str = “字符”；var reg = /正则表达式/；reg.text(str);
（3）compile：编译正则表达式。
（4）exec：检索字符串中指定的值，返回找到的值，并确定其位置。和match不加g的时候的返回值相同，但这种方法加g之后多次执行，第二次执行是在第一次的基础上继续向下查找。查找是一个循环。第三次查找，游标处于字符的最后面，第四次查找是，游标又处于第一位。再次从第一位进行查找。

支持正则表达式的String对象的方法

（1）search：检索与正则表达式相匹配的值。str.search();括号中可以填需要查找的字符或者用正则表达式匹配的字符。返回要找的字符在str中的索引位。
（2）match：找到一个或多个正则表达式的匹配。match返回的是一个类数组，第一位是匹配到的字符，第二位是正则的第一个子表达式匹配到的字符，第三位是第二个子表达式的内容，index：3，是正则匹配到的在源字符上的位置，input位上的是源字符。执行全局匹配时返回的值只有匹配到的值。
（3）replace：替换与正则表达式匹配的子串。字符串在进行相互替换时，只能找到一个需要替换的字符，无法进行全局的字符进行替换。

（4）split：把字符串分割为字符串数组。将字符串按照括号中匹配到的字符进行分割。

正则表达式的元字符

正则表达式的量词（贪心匹配，拿到最多的片段）

①打破贪心匹配的方法：加问号 n+?最多匹配一个，n*?最多匹配一个。n??能取零绝对不取一个。
②var reg = /^abc$/g;以abc开头并且以abc结尾的字符。此处限定只能是abc。
var reg = /^(abc|bcd)$/g;此处限定了匹配字符只能是abc或者bcd。
③?:的作用：reg = /^(?:b|c)\1/g;此时的\1不代表第一个子表达式，?:相当于将该括号中不看作一个子表达式。
④?=x，正向预查：要匹配的字符串，后面必须紧跟着x字符。reg = /cainiao(?=8)/g。匹配到的菜鸟字符后面必须紧跟数字8。
⑤?!,非正向预查：要匹配的字符串，后面不能紧跟着某个字符。

正则表达式对象的属性

lastIndex：游标进行下次查找时所处的位置。可以自己先改动lastIndex的值，让查找的开始为不是零。exec方法全局匹配下的lastIndex的值会改变。不是全局匹配下的lastIndex的值每次查找后都会置零。

$字符的注意点

在替换文本中，$有了特殊含义，即代表说匹配到的字符串，$1代表第一个子表达式，所以如果我们想用$这个字符的话，需要写成$$，此时前一个$符成了转义符号。匹配问号时，需要进行转义。
